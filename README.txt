Per eseguire un test sulla precisione chiamare la funzione test_precision(data_set, p)
  - data_set è una stringa che specifica il file che contiene il data set: deve essere in formato csv con la classificazione come ultimo 	attributo e gli attributi devono essere categorici
  - p è la probabilità con cui uniform_deletion(p, data_set) elimina uniformemente i valori delle celle

i link ai data set usati sono:

 - Car evaluation: http://archive.ics.uci.edu/ml/datasets/Car+Evaluation
 - tic tac toe endgame : http://archive.ics.uci.edu/ml/datasets/Tic-Tac-Toe+Endgame
 - Nursery: http://archive.ics.uci.edu/ml/datasets/Nursery
